# ECEN_430_Tutorials
For this Tutorial to work you need to clone this repo to `~/catkin_ws/src/`:

```
$ cd ~/catkin_ws/src/
$ git clone https://github.com/skipper762/ECEN_430_Tutorials
```

Now we need to compile the code
```
$ cd ~/catkin_ws/
$ catkin_make
```
this should return no errors.
